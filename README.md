# Météolys

Weather PWA developed with Vue.js and [Open-Meteo](https://open-meteo.com).

User can search a city by name or use its actual location.
A local storage is used to save the last city.
French weather alerts are managed by Météo-France.

Charts for forecasts are displayed with [Echarts](https://echarts.apache.org/en/index.html).

[Weather icons](https://github.com/mrdarrengriffin/google-weather-icons).

## For developers

``npm i`` to install all dependencies

``npm run dev`` to start the server

``npm run scss`` to compile scss to css
